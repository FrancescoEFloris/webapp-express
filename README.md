# CONSEGNA DEL PROGETTO:

## PorcheDotto 🐷🎓  

**PorcheDotto** è una porchetteria dall'animo geniale, dove ogni prodotto sembra il risultato di un esperimento scientifico finito molto bene.

Qui si può giocare con nomi simpatici come:

- Panino Formula Segreta;
- Porchetta Quantistica;
- Croccante di laboratorio;
- Combo Premio Nobel.

Categorie possibili:

- Panini
- Piatti pronti
- Contorni
- Salse
- Esperimenti riusciti

Il tono può essere comico, pseudo-scientifico e perfetto per inserire descrizioni prodotto assurde ma memorabili.
